## 📖 About

This repository contains my personal solutions to **334 LeetCode problems**, all implemented in **C++**. Problems span a wide range of topics including arrays, strings, linked lists, trees, graphs, dynamic programming, bit manipulation, and SQL database queries.

Solutions are automatically synced using [LeetSync](https://github.com/isamert/leetcode-sync), keeping the repository up to date with every accepted submission.

---

## 📂 Repository Structure

Each problem lives in its own folder named `<problem-number>-<problem-slug>/`, containing:

```
<problem-number>-<problem-name>/
├── README.md          # Problem statement with examples and constraints
└── <solution>.cpp     # C++ solution
```

**Example:**
```
1-two-sum/
├── README.md          # Problem description
└── two-sum.cpp        # C++ solution
```
